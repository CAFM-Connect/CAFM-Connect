Terms, definitions, and abbreviated terms

For the purpose of this document, the following definitions apply.

3.1 Terms and definitions

3.1.1 asset

item, thing or entity that has a potential or actual value to an organization.

[ISO 19650-1:2017 3.2.8]

3.1.2 attribute

any data relative to the description of a property, group of properties, entity etc.

NOTE: for example, name, definition, etc. [EN ISO 23386, 3.4]

3.1.3 configurable product

a product offered by a manufacturer, that can be configured and ordered based on the offerings of the manufacturer

3.1.4 configured product

a product offered by a manufacturer, that cannot be further configured. It is offered in exact one variant.

3.1.5 data sheet

description of construction objects compliant to a data template [3.1.6]

3.1.6 data template

data structure used to describe the characteristics of construction objects

[EN ISO 23387, 3.3]

3.1.7 Industry Foundation Classes (IFC)

specifies a conceptual data schema and an exchange file format for Building Information Model (BIM) data

[ISO 16739:2013]

3.1.8 information

meaningful data

[EN ISO 9000:2015, 3.8.2]

3.1.9 Information structure

Machine readable schema for storing and transferring data

3.1.10 manufacturer

Person or organization making offsite materials, products, components, and other items.

[ISO 6707-2:2017, 3.6.3]

3.1.11 object

part of the perceivable or conceivable world.

[ISO 12006-2:2015, 3.1.1]

3.1.12 product

tangible outcome of a process

[ISO 6707-3, 3.3.1]

3.1.13 property

inherent or acquired feature of a product or an object

[ISO 6707-1:2017, 9.1.3]

3.1.14 requirement request

a collection of information that describes the construction objects, that are needed to fullfil a specific purpose in the design of a building

3.2 Abbreviated terms

COD Construction Objects Data View Construction Objects Data View is a specific Model View Definition that store the IFC entities required to exchange information through data templates and data sheets.
PDS Product Data Sheets Product Data Sheets are data template containing product information
PDT Product Data Template data template relating to product information
