# cafm-connect
CAFM-Connect is a standard to exchange data of facilities based on IFC4

Was ist CAFM-Connect?
CAFM-Connect ist eine Initiative des CAFM-Rings und steht dem Markt als offene Schnittstelle im Facility-Management zur Verfügung. Mit CC können Daten von Gebäuden zwischen Softwaresystemen abpsrachlos und sicher ausgetauscht werden.

CAFM-Connect basiert auf dem weltweiten IFC-Standard: http://www.buildingsmart-tech.org. CC basiert auf IFC4.

Der IFC-Standard umfasst eine sehr große Anzahl von Elementen, die sehr viele Fachgebiete für Bau- und Betrieb von Immobilien abdecken. Mit einer IFC-View können Teilmengen aus dem gesamten IFC-Sprachraum definiert werden. CC basiert auf insgesamt vier IFC-Views, die eigens für CAFM-Connect definiert wurden. Lesen Sie sich dazu ein um einen Überblick zu dem Konzept der Model View Definition zu erhalten.
