# CAFM-Connect
CAFM-Connect is a standard to exchange data of facilities based on IFC4.

Was ist CAFM-Connect?
CAFM-Connect ist eine Initiative des CAFM-Rings und steht dem Markt als offene Schnittstelle im Facility-Management zur Verfügung. Mit CAFM-Connect können Daten von Gebäuden zwischen Softwaresystemen abpsrachlos und sicher ausgetauscht werden. Weitere Informationen finden Sie hier: http://www.cafm-connect.org.

Eine CAFM-Connect Datei basiert auf dem weltweiten IFC-Standard: http://www.buildingsmart-tech.org. Eine CAFM-Connect Datei basiert auf ifcXML und kann damit leicht und sicher in Softwaresystemen implementiert werden.

Der IFC-Standard umfasst eine sehr große Anzahl von Elementen, die sehr viele Fachgebiete für Bau- und Betrieb von Immobilien abdecken. Mit einer IFC-View können Teilmengen aus dem gesamten IFC-Sprachraum definiert werden. CC basiert auf insgesamt vier IFC-Views, die eigens für CAFM-Connect definiert wurden. Lesen Sie sich dazu ein, um einen Überblick zu dem Konzept der Model View Definition zu erhalten: http://www.buildingsmart-tech.org/specifications/mvd-overview/mvd-overview-summary

Die Inhalte einer CAFM-Connect Datei sind strukturiert auf Basis des CAFM-Connect Katalogs: http://katalog.cafm-connect.org . Unser GitHub Repository zum Katalog finden Sie hier: https://github.com/cafmring/CAFM-Connect-Katalog.

Die Views (MVD's) auf Basis des Katalogs werden im Katalogeditor (http://katalogeditor.cafm-connect.org) editiert und dann hier veröffentlicht: http://katalog.cafm-connect.org/MvdViewer.aspx (dort haben wir derzeit erst ein einfaches Beispiel).

Wie kann ich CAFM-Connect implementieren?
Die hier veröffentlichten Ressourcen stehen den Mitgliedern und Freunden des AK-Technik des CAFM-Rings zur Verfügung. Falls Sie interessiert sind, mit CAFM-Connect ihre Immobilienprozesse zu digitalisieren, dann melden Sie sich bitte bei uns, um die nächsten Schritte zu erfahren.
